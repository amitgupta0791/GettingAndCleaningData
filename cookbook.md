Cookbook - Course Project - Creating Tidy Data
======================

- Output file from the function is tidyfile.txt
- This is comma separated file with extension .txt
- This file will have 180 rows of data
- first column of the file is the subject experiment is being conducted
- second column is the name of the activity being performed.
- next 66 columns have the means of the means and standard deviation of various measurement.
